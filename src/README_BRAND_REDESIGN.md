## 📊 Design Principles

### 60-30-10 Color Ratio
- **60% Neutral** - White, cream, light grays for backgrounds
- **30% Slate** - Text, icons, UI elements
- **10% Accent** - Terracotta and forest green for highlights

### Typography
- **Font**: Inter (professional, modern, UI-optimized)
- **Base Size**: 14px (perfect for data-dense ERP)
- **Weights**: 300, 400, 500, 600, 700

### Spacing
- **Card Padding**: 24px (was 16px)
- **Page Padding**: 32-48px (was 24px)
- **Section Gaps**: 32-40px (was 16-24px)
- **More Breathing Room**: 30-50% increase overall
