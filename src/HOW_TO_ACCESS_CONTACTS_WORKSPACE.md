## 🔍 Where to Find It

The **ContactsWorkspaceV4** is now accessible through the sidebar navigation in the **Agency Module**.

### Option 1: Sidebar Navigation (Primary)
1. **Login** to aaraazi
2. **Select Agency Module** (if prompted)

## 🎯 What You'll See

When you click "Contacts" in the sidebar, you'll see:

### Header Section
- **Title**: "Contacts"
- **Description**: "Manage your contacts and relationships"
- **Stats**: Total, Active, Clients, Commission
- **Add Contact Button** (blue, top-right)

### Main Workspace
- **View Switcher**: Toggle between Table and Grid views
- **Search Bar**: Search by name, phone, email, or notes
- **Quick Filters**: Type, Status, Category
- **Sort Options**: Name, Date, Last Contact, Commission

### Table View (Default)
9 columns showing:
1. Name (with email)
2. Phone
3. Type (Client, Prospect, Investor, Vendor)
4. Category (Buyer, Seller, Tenant, Landlord)
5. Status (Active, Inactive, Archived)
6. Properties (interested count)
7. Deals (transaction count)
8. Commission (total earned)
9. Last Contact (days ago)

### Grid View
Beautiful cards showing:
- Contact avatar
- Name and type
- Phone and email
- Properties interested
- Commission earned
- Last contact date
- Quick actions menu (3 dots)

## ⚡ Quick Actions

### From the Workspace:
- **Add Contact**: Click the blue "Add Contact" button (top-right)
- **View Contact**: Click any card or table row
- **Edit Contact**: Click the 3-dot menu → Edit
- **Select Multiple**: Use checkboxes for bulk operations
- **Filter**: Click filter icon to show/hide filters
- **Sort**: Use dropdown to change sorting

### Bulk Operations (when items selected):
- **Export**: Export selected contacts
- **Archive**: Archive multiple contacts
- **Delete**: Delete multiple contacts (with confirmation)

## 📊 Data Displayed

### Contact Information:
- **Name**: Full contact name
- **Email**: Email address (if available)
- **Phone**: Contact phone number
- **Type**: Client, Prospect, Investor, or Vendor
- **Category**: Buyer, Seller, Tenant, Landlord, or Both
- **Status**: Active, Inactive, or Archived

### Relationship Data:
- **Properties**: Count of interested properties
- **Deals**: Count of completed transactions
- **Commission**: Total commission earned from this contact
- **Last Contact**: How many days since last interaction

### Metadata:
- **Created**: When contact was added
- **Updated**: Last modification date
- **Agent**: Who manages this contact

## 🔐 Access Control

### Who Can See What:

**Agents**:
- See only their own contacts
- Can add new contacts
- Can edit their contacts
- Can archive/delete their contacts

**Admins**:
- See ALL contacts across the agency
- Can manage any contact
- Can perform bulk operations
- Full access to all features

## 💡 Usage Tips

### Best Practices:
1. **Use Type wisely**: 
   - Client = Active revenue source
   - Prospect = Potential client
   - Investor = High-value investment client
   - Vendor = Service providers

2. **Track Categories**:
   - Buyer/Seller for sale transactions
   - Tenant/Landlord for rentals
   - "Both" for contacts who buy AND sell

3. **Keep Last Contact Updated**:
   - Helps prioritize follow-ups
   - Shows relationship health
   - Sort by "Last Contact" to find cold leads

4. **Use Commission Tracking**:
   - See your most valuable clients
   - Sort by commission to prioritize
   - Track revenue per contact

5. **Bulk Operations**:
   - Archive inactive contacts quarterly
   - Export for marketing campaigns
   - Clean up duplicate entries

## 🐛 Troubleshooting

### "I don't see the Contacts menu item"
- Make sure you're in the **Agency Module**
- Check that you're logged in
- Look under "Relationships" section in sidebar

### "No contacts showing"
- Click "Clear all filters" if filters are active
- Check that you have contacts in the system
- Try adding a new contact first

### "Can't see all contacts"
- Agents see only their contacts
- Admins see all contacts
- Check pagination (bottom of page)

### "Edit contact doesn't work"
- This feature is still being built
- For now, use the old CRM to edit
- Edit functionality coming soon
