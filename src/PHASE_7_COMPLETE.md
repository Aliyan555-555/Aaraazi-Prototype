## 🎓 Best Practices Established

### 1. Always use semantic colors:
```tsx
// ✅ DO
<div className="bg-neutral-50 text-slate-700">

// ❌ DON'T
<div className="bg-gray-100 text-gray-900">
```

### 2. Never use Tailwind typography classes:
```tsx
// ✅ DO - Let CSS handle it
<h1>Dashboard</h1>

// ❌ DON'T - Override CSS defaults
<h1 className="text-2xl font-bold">Dashboard</h1>
```

### 3. Always use StatusBadge for statuses:
```tsx
// ✅ DO
<StatusBadge status="Active" size="sm" />

// ❌ DON'T
<span className="px-2 py-1 bg-green-100 rounded">Active</span>
```

### 4. Always use chartColors.ts for charts:
```tsx
// ✅ DO
import { PRIMARY_SEQUENCE } from '../lib/chartColors';
<Line stroke={PRIMARY_SEQUENCE[0]} />

// ❌ DON'T
<Line stroke="#10b981" />
```

### 5. Always use semantic toast variants:
```tsx
// ✅ DO
toast.success('Success!');
toast.error('Error!');

// ❌ DON'T
toast('Success!');
```
