## Quick Reference

### Contact Types
| Type | Role in Transactions | Tracked Metrics |
|------|---------------------|-----------------|
| **Buyer** | Purchases properties | Properties owned, Total spent |
| **Seller** | Sells properties | Properties sold, Total received |
| **Tenant** | Rents properties | Properties rented, Rent paid |
| **Landlord** | Leases properties | Properties leased, Rent received |
| **Investor** | Co-invests in properties | Investment value, ROI % |

### Key Features
- **Multi-Type Contacts**: Single contact can be Buyer + Investor + Landlord simultaneously
- **Complete History**: All transactions, properties, and interactions preserved
- **Bidirectional Links**: Navigate between contacts, transactions, and properties
- **Rich Interactions**: Phone, email, meetings, viewings all logged
- **Financial Tracking**: Total value transacted, payments, commission generated

### Integration Points
1. **Leads**: Convert successful leads to contacts
2. **Transactions**: Contacts play roles in all cycles
3. **Properties**: Ownership and rental relationships tracked
4. **Financials**: Payment and commission tracking
5. **Documents**: Legal and financial docs linked to contacts

### Contact Lifecycle
- **Creation**: Lead conversion (primary) or manual entry
- **Active**: Ongoing transactions and interactions
- **Relationship**: Multiple transactions over time
- **Portfolio**: Property ownership tracked
- **Persistence**: Never deleted, complete history preserved
