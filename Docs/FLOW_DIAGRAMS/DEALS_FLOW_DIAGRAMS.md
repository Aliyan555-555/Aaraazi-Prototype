## Quick Reference

### Deal Stages
| Stage | Description | Typical Duration | Next Action |
|-------|-------------|------------------|-------------|
| 1. Lead | Initial interest | 1-3 days | Qualify buyer |
| 2. Qualify | Serious interest | 2-5 days | Present proposal |
| 3. Proposal | Offer presented | 3-7 days | Negotiate terms |
| 4. Negotiation | Terms discussion | 5-14 days | Close deal |
| 5. Close | Won or Lost | Terminal | Create transaction |

### Deal Types
- **Sale Deal**: Property purchase transaction
- **Rental Deal**: Property lease transaction
- **Single-Agent**: One agent handles both sides (100% commission)
- **Dual-Agent**: Two agents collaborate (50/50 commission split)

### Commission Structure
| Deal Structure | Total Commission | Agent 1 (Listing) | Agent 2 (Buyer's) |
|----------------|------------------|-------------------|-------------------|
| **Single-Agent** | 2% of sale | 100% (2%) | N/A |
| **Dual-Agent** | 2% of sale | 50% (1%) | 50% (1%) |

### Key Integrations
1. **Properties**: Deals linked to specific properties
2. **Contacts**: Client/buyer relationships
3. **Transactions**: Auto-create Sell/Rent Cycle on win
4. **Leads**: Convert leads to deals
5. **Financials**: Commission tracking
6. **Sharing System**: Cross-agent collaboration

### Deal Outcomes
- **Closed Won**: Deal successful â†’ Create transaction
- **Closed Lost**: Deal failed â†’ Archive with reason, learn
- **Active**: Still in pipeline â†’ Progress through stages
