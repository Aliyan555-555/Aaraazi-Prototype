## Quick Reference

### Portfolio Components

| Component | Description | Key Metrics |
|-----------|-------------|-------------|
| **Agency Inventory** | 100% owned properties for resale | Cost, Target, Holding, ROI |
| **Investment Syndication** | Multi-investor co-ownership | Share %, Current value, ROI |
| **Investor Management** | Track individual investors | Total invested, Current value, ROI |
| **Performance Analytics** | Portfolio analysis | ROI by type, location, period |

### Inventory Tracking

- **Acquisition**: Purchase Cycle completion
- **Holding**: Marketing, maintenance, showing
- **Exit**: Sell Cycle completion
- **Profit**: Sale price - (Acquisition + Holding costs)
- **ROI**: (Profit / Total Cost) Ã— 100

### Syndication Model

- **Structure**: Multiple investors, defined % shares
- **Ownership**: Proportional to investment amount
- **Returns**: Distributed by ownership %
- **Management**: Agency coordinates property & investors
- **Exit**: Sale triggers profit distribution

1. **Properties**: All portfolio properties tracked

3. **Contacts**: Investors as special contact type
4. **Financials**: Profit/loss tracking, investor distributions
5. **Reports**: Portfolio performance reports
