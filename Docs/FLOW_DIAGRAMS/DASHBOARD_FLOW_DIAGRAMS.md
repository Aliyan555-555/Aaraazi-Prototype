## Quick Reference

### Dashboard Components

| Component | Purpose | Key Features |
|-----------|---------|--------------|
| **Action Center** | Daily tasks | Overdue, Today, Upcoming items |
| **Intelligence Panel** | AI insights | Alerts, Insights, Recommendations |
| **Quick Actions** | Fast access | 8 common actions |
| **Performance Metrics** | KPIs | 20+ real-time metrics |
| **Recent Activity** | Updates | Live system activity |

### Quick Actions (8 Total)

1. Add Property
2. Add Contact
3. Add Lead
4. Add Deal
5. Start Sell Cycle
6. Log Call
7. Schedule Meeting
8. Record Payment

### Metric Categories

**Revenue**: Total, Commission, Monthly Target, YTD  
**Sales**: Deals Closed, Avg Deal, Win Rate, Cycle Time  
**Pipeline**: Leads, Deals, Properties, Conversion  
**Financial**: Profit, Margin, Outstanding, Cash Flow  
**Team**: Top Agent, Team Deals, Response, Satisfaction  

### Role-Based Access

**Admin**:
- All agents' data visible
- System-wide metrics
- Team performance
- Can reassign items

**Agent**:
- Personal data only
- Individual metrics
- Own tasks/leads
- Cannot view others
