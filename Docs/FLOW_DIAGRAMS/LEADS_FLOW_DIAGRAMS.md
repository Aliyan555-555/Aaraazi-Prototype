## Quick Reference

### Lead Stages
| Stage | Description | Typical Duration | SLA |
|-------|-------------|------------------|-----|
| 1. New | Just created | Day 0-1 | 24h (High) |
| 2. Contacted | First touch made | Day 1-3 | 48h (Medium) |
| 3. Qualified | Serious interest | Day 3-10 | 72h (Normal) |
| 4. Negotiation | Deal discussion | Day 10-18 | 3-5 days |
| 5. Closed | Won or Lost | Day 18-21 | Terminal |

### Key Actions
- **Add Lead**: Manual entry, Quick capture, Property interest
- **Log Activity**: Phone, Email, Meeting, Property viewing
- **Schedule Follow-up**: Set reminders, Track touchpoints
- **Progress Stage**: Move through pipeline
- **Convert to Client**: Create Contact + Transaction
- **Mark as Lost**: Record reason, Archive lead

### Integration Points
1. **Properties**: Match to lead requirements
2. **Contacts**: Convert successful leads
3. **Transactions**: Auto-create on conversion
4. **Dashboard**: Follow-up alerts, Performance metrics

### 21-Day Milestones
- **Day 0-3**: First contact, Initial qualification
- **Day 3-7**: Property matching, Follow-ups
- **Day 7-14**: Property showings, Interest confirmation
- **Day 14-21**: Negotiation, Decision time
- **Day 21+**: Convert or lose
