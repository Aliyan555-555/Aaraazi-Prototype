## âœ… Quality Checklist

When creating any design, verify:

### Color Usage
- [ ] 60% neutral (white, cream, light grays)
- [ ] 30% slate (text, UI elements)
- [ ] 10% accent (terracotta, forest green)
- [ ] No pure black (#000000)
- [ ] Terracotta used sparingly
- [ ] Success states use forest green

### Spacing
- [ ] Page padding: 32-48px
- [ ] Section gaps: 32-40px
- [ ] Card padding: 24px
- [ ] Element gaps: 16-24px
- [ ] Form field gaps: 24px
- [ ] Feels generous, not cramped

### Typography
- [ ] Inter font is used
- [ ] Headings: semibold (600)
- [ ] Body: normal (400)
- [ ] Labels: medium (500)
- [ ] Proper heading hierarchy (h1â†’h2â†’h3)
- [ ] Line heights: 1.5 for body, 1.25 for headings

### Accessibility
- [ ] Color contrast â‰¥ 4.5:1 (WCAG AA)
- [ ] Focus states visible
- [ ] Interactive elements â‰¥ 44x44px
- [ ] Text readable at all sizes
- [ ] Links distinguishable

### Visual Polish
- [ ] Consistent border radius (10px default)
- [ ] Subtle shadows on cards
- [ ] Smooth transitions (150ms)
- [ ] No visual glitches
- [ ] Professional appearance
